## HTML tags

### <h2 style="color:green">❖ Text tags </h2>

- **Headers**
  _Headers go from h1 to h6_

<h1>H1 - Header</h1>
<h2>H2 - Header</h2>
<h3>H3 - Header</h3>
<h4>H4 - Header</h4>
<h5>H5 - Header</h5>
<h6>H6 - Header</h6>

```html
<h1>H1 - Header</h1>
<h2>H2 - Header</h2>
<h3>H3 - Header</h3>
<h4>H4 - Header</h4>
<h5>H5 - Header</h5>
<h6>H6 - Header</h6>
```

---

- **Paragraph**
  _paragraph doesnt recognize empty spaces if its more than one_

```html
<p>Never give up on your dreams.</p>
```
